Positive Change

Website

PositiveChange.com is a website that contains various media and text to motivate users, elevate moods, and encourage positive emotions.

PositiveChange.com will offer mainly text-based motivation quotes, photos, and links to videos related to motivation and uplifting thoughts. The website is meant to either change how the user feels or empathize with them. It should not be considered a library for motivational quotes, but rather an experience meant to improve the user's state of mind.
